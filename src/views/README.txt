本文件夹下包含了管理员前端各个功能页面的功能调用及实现
1. acl是权限管理主要包含：
    1.1 menu 菜单管理
    1.2 role 角色管理
    1.3 user 用户管理
2. cms/banner是轮播图管理
3. dashboard是管理员中心页面
4. edu是与课程相关的管理
    4.1 course 课程管理
    4.2 subject课程分类管理
    4.3 teacher讲师管理
5. layout是登出功能
6. login是登入功能
7. order是订单管理
8. statistics/daily是统计管理
9. 以上未说明文件均为框架文件
